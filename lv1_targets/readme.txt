本フォルダにはレベル1用のターゲット認識器を可視化した画像群が保存されています．
レベル1用のターゲット認識器はいずれも，
MNIST訓練データ60,000枚を用いて学習した10クラス識別器です．

各画像の解像度は 512x512 で，横軸が特徴量の1次元目(x1)，縦軸が特徴量の2次元目(x2)を表します．
各次元の値は -1 から 1 までであり，左端が x1 = -1 ，右端が x1 = 1 に対応します．
同様に，上端が x2 = 1 に，下端が x2 = -1 に対応します．

色はクラスラベルIDを表しています．
対応関係は次の通りです．

  - 0: 青
  - 1: 明るい水色
  - 2: 緑
  - 3: 黄色
  - 4: 赤
  - 5: ピンク
  - 6: 黒
  - 7: 暗い水色
  - 8: オレンジ
  - 9: 紫
